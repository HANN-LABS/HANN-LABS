커뮤니티 운영정책

한워크스 디스코드 커뮤니티는 아래와 같은 행위를 금지하며, 위반 시 수위에 따라 직접 판단하여 처벌합니다.

v.0.0.1

금지 행위

  • 심한 욕설을 사용하여 남을 비난, 비판한 경우
  • 서버 내의 유해한 정보를 유포한 경우 (해킹이더라도 처벌)
  • 서버 운영에 심각한 피해를 끼쳐 운영에 문제가 생긴 경우
  • 음란물 또는 고어물 등의 유해한 정보를 유포한 경우
  • 불법 프로그램을 판매, 배포한 경우
  • 디스코드 공식 규정(ToS 등)을 위반한 경우
  • 도배 및 핑을 남용한 경우
  • 소수 혹은 다수의 유저끼리 모여 특정 유저에게 소외감을 준 경우
  • 스태프, 운영진을 사칭한 경우
  • 자유홍보 채널이 아닌 곳에서 홍보한 경우
  • 개인 메시지로 특정 서버나 서비스를 홍보한 경우
  • 타인의 개인정보를 공유하거나 유포한 경우
  • 커뮤니티의 주제와 목적에 맞지 않는 민감 주제로 분쟁을 유발할 수 있는 경우
  • 본 정책에 명시된 내용 이외에도 운영진 판단하에 제재가 필요할 경우

브랜드 파트너십 운영정책

v.0.0.2

파트너십 조건

  • 한워크스 디스코드 커뮤니티 운영 정책에 위배되지 않는 서버
  • 단순 인원 보충이 아닌 명분과 가치 전달을 할 수 있는 발전 가능성이 있는 서버
  • 계획과 목표가 뚜렷한 서버
  • 협력을 통한 상호발전이 가능한 서버
  • ※ 인원 제한은 없으나 위 조건 중 2개 이상을 충족해야 합니다.

파트너십 해지 기준

  • 한워크스 디스코드 커뮤니티 운영 정책에 위배된 경우
  • 서버장이 바뀐 경우
  • 서버명이 바뀐 경우
  • 서버에 2개월 이상 활동이 없어 유령서버가 된 경우
